# Problema de Negocio

## Contexto

Imagine uma pequena empresa de servicos B2B que recebe diariamente uma planilha CSV com novos clientes interessados em contratar uma solucao.

Esses clientes podem vir de formularios, campanhas, indicacoes, eventos ou parceiros. O arquivo chega com dados variados: nome, email, telefone, documento, cidade, origem do lead e tipo de interesse.

Apesar de parecer simples, esse processo consome tempo todos os dias.

## Antes: Processo Manual

No processo atual, uma pessoa precisa:

- localizar o CSV recebido;
- abrir o arquivo em uma planilha;
- conferir campos obrigatorios;
- corrigir emails mal formatados;
- padronizar nomes, telefones e cidades;
- identificar linhas duplicadas;
- consultar manualmente informacoes adicionais;
- classificar clientes por regra de negocio;
- salvar uma nova versao do arquivo;
- montar um relatorio;
- avisar a equipe responsavel;
- mover o arquivo para uma pasta de processados.

Esse fluxo depende de memoria, atencao e disciplina. Quando o volume cresce, os erros tambem crescem.

## Desperdicios Eliminados

A automacao proposta elimina desperdicios como:

- tempo gasto com tarefas repetitivas;
- retrabalho por erro de digitacao ou padronizacao;
- analise manual de duplicidades;
- criacao recorrente de relatorios;
- perda de arquivos processados;
- falta de visibilidade sobre falhas;
- dependencia de uma unica pessoa para executar a rotina.

## Depois: Processo Automatizado

No processo automatizado, o operador coloca o CSV na pasta de entrada e executa o fluxo.

O sistema:

- le o arquivo recebido;
- valida campos obrigatorios;
- separa registros validos e invalidos;
- limpa e padroniza os dados;
- remove duplicidades;
- enriquece registros usando uma API simulada;
- aplica regras de negocio;
- gera arquivos de saida;
- cria relatorios;
- registra logs;
- prepara uma notificacao;
- arquiva o arquivo original.

O trabalho humano deixa de ser operacional e passa a ser de supervisao.

## Resultado Esperado

Ao final do processo, a empresa tem:

- dados mais confiaveis;
- tempo operacional reduzido;
- historico do processamento;
- relatorio objetivo;
- arquivos organizados;
- base preparada para integracoes futuras.

## Valor Para o Cliente

O valor principal nao esta em usar uma ferramenta especifica. Esta em provar que uma rotina real pode ser analisada, organizada e automatizada com criterio.

Esse tipo de automacao reduz custo operacional, melhora qualidade dos dados e libera pessoas para atividades de maior valor.
